# React router dom

[View template - beamish-parfait-98cd2a.netlify.app](https://beamish-parfait-98cd2a.netlify.app/)

- აირჩიე რომელიმე მხატვრული პერსონაჟი და შექმენი აპლიკაცია react-router-dom ბიბლიოთეკის გამოყენებით
- აპლიკაციაში უნდა იყოს მთავარი გვერდი, პერსონაჟის ფოტოთი და სახელით, /about გვერდი, სადაც პერსონაჟზე ზოგადი ინფორმაცია იქნება მოთავსებული და /:factId გვერდი, სადაც JavaScript ობიექტიდან იქნება შერჩეული ფაქტი კონკრეტული id-ით
- თუ ფაქტი id - ით არ არსებობს, გვერდზე უნდა დაეწეროს, რომ ფაქტი ვერ მოიძებნა

## Solution

- პერსონაჟი რეალური ცხოვრებიდავ ავიღე
- შევქმენი სამი გვერდი - მთავარი, შესახებ და ფაქტები
  - მთავარ გვერდზე ფოტო, სახელი და ღილაკი გავნათავსე
  - შესახებ გვერდზე - ფოტო და მოკლე ინფორმაცია ვიკიპედიიდან
  - ფაქტების გვერდზე - ორი ფაქტი უამრავიდან, რაც ტიურინგის სახელს უკავშირდება
- ინფორმაცია მოდის json ფილიდან, რომელიც პროექტში არის განთავსებული
- Not found page გაწერილია როტერის path-ით, "\*"
- ხოლო ფაქტი ვერ მოიძებნა გვერდი, ჩემით ვერ გავაკეთე და დახმარება AI-ის ვთხოვე. მგონი ცუდი არ გამოვიდა
- სტილებისთვის კი გამოვიყენე tailwindCSS პაკეტი
